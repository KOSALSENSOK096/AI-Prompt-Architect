# AI Prompt Architect

A powerful AI-powered application for generating, managing, and optimizing prompts for various AI models.

## Features

- AI Prompt Generation
- Prompt Translation
- UI Design Assistance
- Image Enhancement
- Premium Chat Features

## Setup and Installation

1. Clone the repository:
```bash
git clone https://github.com/KOSALSENSOK096/AI-Prompt-Architect.git
cd AI-Prompt-Architect
```

2. Install dependencies:
```bash
npm install
```

3. Create a `.env.local` file in the root directory and add your Gemini API key:
```bash
GEMINI_API_KEY=your_api_key_here
```

4. Start the development server:
```bash
npm run dev
```

The application will be available at http://localhost:5174 (or another port if 5174 is in use)

## Environment Variables

Create a `.env.local` file in the root directory with the following:

```
GEMINI_API_KEY=your_api_key_here
```

**Important:** Never commit your `.env.local` file or expose your API key publicly.
